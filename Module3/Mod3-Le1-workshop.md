## Часть 2: Что такое JavaScript и зачем он дизайнеру? (20 минут)

### JavaScript — язык браузера

**Теория:** 
- JavaScript создан специально для работы в браузере
- Позволяет "оживлять" статичные макеты
- Альтернативы: TypeScript, CoffeeScript (транспилируются в JS)

**Визуальная аналогия:** 
Если HTML — скелет страницы, CSS — её одежда и макияж, то JavaScript — это нервная система, которая заставляет реагировать на прикосновения.

### Возможности JS для дизайнеров

**Демонстрация примеров из портфолио выпускников:**
1. Генерация повторяющихся элементов (каталоги, галереи)
2. Интерактивные веб-плакаты с реакцией на курсор
3. Динамическое изменение стилей (темы, анимации)
4. Валидация форм с визуальным фидбеком

## Часть 5: Стили кодирования для дизайнеров

### Правила именования переменных

**CamelCase для дизайнеров:**
```javascript
// Хорошо:
let primaryColor = '#3366ff';
let animationDuration = 300;
let isDarkTheme = true;

// Плохо:
let primary_color = '#3366ff'; // snake_case (не принято в JS)
let primary-color = '#3366ff'; // дефисы (ошибка!)
let pc = '#3366ff'; // непонятное сокращение
```

### Константы в дизайн-системах

```javascript
// Константы для дизайн-системы

const MAX_WIDTH = 1200;
```

## Часть 6: Интерактивный мини-проект

### Создаём "живую" визитку

```html
<!DOCTYPE html>
<html>
<head>
    <style>
        .card {
            border: 2px solid #333;
            padding: 20px;
            max-width: 300px;
            transition: all 0.3s;
        }
        .dark {
            background: #222;
            color: white;
        }
    </style>
</head>
<body>
    <div class="card" id="businessCard">
        <h2 id="name">Анна Дизайнер</h2>
        <p id="role">Веб-дизайнер</p>
        <button onclick="toggleTheme()">Сменить тему</button>
        <button onclick="updateCard()">Обновить данные</button>
    </div>

    <script>
        // Данные визитки
        let designerName = 'Анна Дизайнер';
        let designerRole = 'Веб-дизайнер';
        let isDarkTheme = false;
        
        // Функция смены темы
        function toggleTheme() {
            let card = document.getElementById('businessCard');
            isDarkTheme = !isDarkTheme;
            
            if (isDarkTheme) {
                card.classList.add('dark');
                console.log('Тёмная тема включена');
            } else {
                card.classList.remove('dark');
                console.log('Светлая тема включена');
            }
        }
        
        // Функция обновления данных
        function updateCard() {
            designerName = prompt('Введите ваше имя:', designerName);
            designerRole = prompt('Введите вашу роль:', designerRole);
            
            document.getElementById('name').textContent = designerName;
            document.getElementById('role').textContent = designerRole;
            
            console.log(`Визитка обновлена: ${designerName}, ${designerRole}`);
        }
    </script>
</body>
</html>
```
