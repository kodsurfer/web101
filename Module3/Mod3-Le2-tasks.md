### **Задача №3 / Изменение стилей через JS**

### **Задача №8 / Комплексная задача: Форма с шагами (мастер)**
Создайте простой мастер из двух шагов:
1.  **Шаг 1:** Поле для ввода имени и кнопка "Далее".
2.  **Шаг 2:** (Изначально скрыт) Текст "Привет, {имя}! Выберите цвет фона:" и три кнопки с цветами. Кнопка "Назад".

**Логика:**
*   При вводе имени и нажатии "Далее" — скрывается Шаг 1, показывается Шаг 2, в тексте подставляется имя.
*   При клике на кнопку цвета — фон всей страницы меняется на выбранный цвет.
*   При нажатии "Назад" — возвращается Шаг 1, Шаг 2 скрывается, фон сбрасывается.

**Подсказка:** Используйте `display: none/block` для скрытия/показа шагов (`<div class="step">`). Храните введенное имя в переменной. Для сброса фона — `document.body.style.backgroundColor = ''`.
