## Часть 2: Основы JavaScript - сравнения и вывод данных

### Операторы сравнения
JavaScript предоставляет несколько операторов для сравнения значений:

```javascript
// Базовые операторы сравнения
a > b    // больше
a < b    // меньше
a >= b   // больше или равно
a <= b   // меньше или равно
a == b   // равно (нестрогое сравнение)
a != b   // не равно (нестрогое сравнение)
a === b  // строгое равно (сравнивает тип и значение)
a !== b  // строгое не равно
```

### Работа с console.log()
Команда `console.log()` - ваш лучший друг при отладке:

```javascript
let a = 10;
let b = 5;

console.log(a > b);    // true
console.log(a < b);    // false
console.log(a == 10);  // true
console.log(a != b);   // true
```

**Интерактив:** Откройте инструменты разработчика (F12) → Console и попробуйте выполнить эти команды

## Часть 3: Типы данных и их сравнение

### Булевый тип (Boolean)
- `true` - истина
- `false` - ложь

### Сравнение чисел и строк
```javascript
// Сравнение чисел
console.log(5 > 3);  // true

// Сравнение строк
console.log("Б" > "А");  // true (по Unicode)
console.log("apple" > "banana");  // false (сравнение посимвольно)
```

### Важное правило сравнения строк
JavaScript сравнивает строки по таблице Unicode, а не по алфавиту:
- Латинская "A" = 41 (в шестнадцатеричной системе)
- Латинская "a" = 61
- Кириллическая "А" имеет другое значение!

**Пример:** `"Яблоко" > "Абрикос"` - сравнение начинается с первых символов "Я" и "А"

## Часть 4: Строгое vs нестрогое сравнение

### Ключевые отличия:
```javascript
// Нестрогое сравнение (==)
console.log(5 == "5");    // true (преобразует типы)
console.log(null == undefined);  // true

// Строгое сравнение (===)
console.log(5 === "5");   // false (разные типы)
console.log(null === undefined); // false

// Отрицание
console.log(5 != "5");    // false (нестрогое)
console.log(5 !== "5");   // true (строгое)
```

**Практическое применение:** Используйте строгое сравнение, чтобы избежать неожиданных ошибок!

## Часть 5: Условные конструкции

### Базовый синтаксис if/else
```javascript
// Простое условие
if (условие) {
    // выполняется, если условие истинно
}

// Полная конструкция
if (условие) {
    // выполняется, если условие истинно
} else if (другоеУсловие) {
    // выполняется, если первое ложно, а второе истинно
} else {
    // выполняется, если все условия ложны
}
```

### Практический пример
```javascript
let year = prompt("В каком году появился стандарт ECMAScript 6?");

if (year == 2015) {
    alert("Верно! ES6 был выпущен в 2015 году");
} else if (year > 2015) {
    alert("Это позже, чем 2015 год");
} else {
    alert("Это раньше, чем 2015 год");
}
```

**Интерактив:** Попробуйте этот код в браузере с разными значениями

## Часть 6: Тернарный оператор

### Альтернатива if/else для простых условий
```javascript
// Синтаксис: условие ? значение_если_истина : значение_если_ложь

let age = 20;
let message = age >= 18 ? "Взрослый" : "Несовершеннолетний";

// Эквивалентно:
let message;
if (age >= 18) {
    message = "Взрослый";
} else {
    message = "Несовершеннолетний";
}
```

**Применение:** Часто используется в современных фреймворках (React, Vue)

## Часть 7: Логические операторы

### Комбинирование условий
```javascript
// И (&&) - оба условия должны быть истинны
if (age >= 18 && age <= 65) {
    console.log("Работоспособный возраст");
}

// ИЛИ (||) - хотя бы одно условие должно быть истинно
if (day === "Суббота" || day === "Воскресенье") {
    console.log("Выходной день");
}

// НЕ (!) - инвертирует значение
if (!isLoggedIn) {
    console.log("Пожалуйста, авторизуйтесь");
}
```

### Приоритет операторов
1. `()` - скобки (самый высокий приоритет)
2. `!` - отрицание
3. `&&` - И
4. `||` - ИЛИ

## Часть 8: Типы данных в JavaScript

### Простые (примитивные) типы:
```javascript
// Строка (String)
let name = "Анна";

// Число (Number)
let age = 25;

// Булево значение (Boolean)
let isStudent = true;

// Null и Undefined
let emptyValue = null;
let undefinedValue;

// BigInt (для больших чисел)
let bigNumber = 1234567890123456789012345678901234567890n;
```

### Сложные типы:
```javascript
// Массив (Array)
let colors = ["красный", "зеленый", "синий"];
console.log(colors[0]); // "красный"

// Объект (Object)
let user = {
    name: "Иван",
    age: 30,
    isAdmin: true
};
console.log(user.name); // "Иван"
```
